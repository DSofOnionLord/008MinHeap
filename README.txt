008MinHeap
==========
1) 입력 데이터를 읽어 내림차순으로 정리하시오. 입력 데이터는 다음과 같은 형식을 갖는다.

%cat input.txt
n				// total # of nodes, where n = 10
3   8  12   5   17   23	// store in the order of a complete binary tree
78  5   7   29	


①  입력 데이터를 읽어 min heap을 구성하고 complete binary tree의 노드 순서로 출력하시오.


②  위의 ①에서 구성한 min heap을 사용하여 내림차순으로 출력하시오.
